📘 Comandos Disponibles

📊 Generales

  • /uptime — Muestra el estado actual del bot
  • /status — Verifica si el bot está online
  • /help — Muestra todos los comandos disponibles.
  • /userinfo — Muestra información detallada de un usuario.
  • /avatar — Muestra el avatar de cualquier miembro.
  • /roles — Lista todos los roles del servidor.
  • /ping — Verifica que Abyssus esté activo.
  • /ia — Abyssus responde con IA avanzada.
  • /invite — Muestra el enlace para invitar a Abyssus.

🪙 Economía
